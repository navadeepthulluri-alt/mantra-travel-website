## Project Overview
This project is a fully responsive, static frontend **Travel Website** developed for the MANTRA 2026 Summer School curriculum. It features clean semantic HTML5 structures, structured CSS3 layouts, basic SEO optimization strategies, and interactive JavaScript functionality.

### Features Implemented:
1. **5 Complete Pages:** Home (`index.html`), About (`about.html`), Packages (`packages.html`), Gallery (`gallery.html`), and Contact (`contact.html`).
2. **Responsive Design:** Utilizes CSS Media Queries to provide seamless layout transitions across desktop, tablet, and mobile displays.
3. **Basic SEO Optimization:** Integrated structural `<title>` tags, meta descriptions, localized keywords, hierarchical headers (`<h1>`-`<h3>`), and informative `alt` text parameters on all active images.
4. **JavaScript Interactivity:** Features a dynamic mobile hamburger navigation drawer and client-side form validation mechanisms on the booking window.

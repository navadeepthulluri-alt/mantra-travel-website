## Project Overview

This project is a responsive frontend **Travel Website** created as part of the **MANTRA 2026 Summer School Frontend Website Assignment**. The website is designed using HTML5, CSS3, and JavaScript with a clean interface, responsive layout, basic SEO optimization, and interactive features for a better user experience.

### Features Implemented

1. **Five Complete Pages:** Home (`index.html`), About (`about.html`), Packages (`packages.html`), Gallery (`gallery.html`), and Contact (`contact.html`).
2. **Responsive Design:** Optimized for desktop, tablet, and mobile devices using CSS Flexbox, Grid, and Media Queries.
3. **Basic SEO:** Includes unique page titles, meta descriptions, meta keywords, semantic heading structure (`<h1>`–`<h3>`), and descriptive image alt text.
4. **JavaScript Functionality:** Mobile navigation menu, contact form validation, and email format validation.
5. **Modern UI:** Simple navigation bar, hero section, package cards, gallery, booking form, and footer with a consistent color theme.
